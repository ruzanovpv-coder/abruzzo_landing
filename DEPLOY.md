# Как запустить CasaAbruzzo как Telegram Mini App

## Что нужно сделать один раз

### Шаг 1 — Создать бота в Telegram
1. Открой Telegram, найди **@BotFather**
2. Напиши `/newbot`
3. Придумай имя бота (например: `CasaAbruzzo`) и юзернейм (например: `CasaAbruzzoBot`)
4. Сохрани полученный **токен** — он выглядит как `7123456789:ABCDefGhIjKl...`

### Шаг 2 — Разместить сайт в интернете (бесплатно)

**Вариант A — GitHub Pages (рекомендую):**
1. Создай аккаунт на [github.com](https://github.com)
2. Создай новый репозиторий (например `casaabruzzo`)
3. Загрузи туда все файлы из папки `abruzzo_landing/`
4. Зайди в `Settings → Pages → Source: main branch / root`
5. Сайт появится по адресу: `https://ИМЯ.github.io/casaabruzzo/landing_tg.html`

**Вариант B — Vercel (ещё проще):**
1. Зайди на [vercel.com](https://vercel.com), войди через GitHub
2. Нажми "New Project", выбери репозиторий
3. Деплой займёт 30 секунд, получишь адрес вида `casaabruzzo.vercel.app`

> ⚠️ Telegram требует **HTTPS**. GitHub Pages и Vercel дают HTTPS автоматически.

### Шаг 3 — Подключить сайт к боту
Напиши @BotFather:
```
/setmenubutton
```
Выбери бота → введи URL твоего сайта (например `https://ИМЯ.github.io/casaabruzzo/landing_tg.html`) → введи текст кнопки, например `🏡 Посмотреть квартиру`

### Шаг 4 — Добавить кнопку в канал
1. Зайди в настройки своего Telegram-канала
2. В поле "Ссылка на обсуждение" или через закреплённое сообщение добавь кнопку со ссылкой на бота
3. Гости нажимают → открывается твой сайт прямо внутри Telegram

---

## Как бот получает заявки на бронирование

Когда гость заполняет форму и нажимает "Подтвердить бронирование", сайт отправляет боту JSON:
```json
{
  "type": "booking_request",
  "checkin": "2026-07-01",
  "checkout": "2026-07-08",
  "guests": "2 гостя",
  "name": "Иван Петров",
  "phone": "+7 999 123-45-67",
  "email": "ivan@mail.ru"
}
```

Чтобы бот пересылал это тебе — нужен простой скрипт (Python, ~20 строк).
Когда будешь готов — скажи, напишем вместе.

## Итого: что получает гость

```
Telegram канал → кнопка "Посмотреть квартиру"
    ↓
Открывается landing_tg.html прямо в Telegram
    ↓
Смотрит фото, читает описание, выбирает даты
    ↓
Нажимает нативную красную кнопку снизу
    ↓
Заполняет форму → подтверждает
    ↓
Бот присылает тебе уведомление с деталями заявки
```
