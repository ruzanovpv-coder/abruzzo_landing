# 🏡 CasaAbruzzo — Landing Page

Сайт-лендинг для сдачи апартаментов в Монтесильвано, Абруццо (Италия).
Работает как обычный сайт и как **Telegram Mini App**.

## Файлы

| Файл | Описание |
|------|----------|
| `landing_tg.html` | Основная версия — для Telegram Mini App |
| `landing_v5.html` | Веб-версия — для прямой ссылки в браузере |
| `admin.html` | Панель хозяина — цены, контакты, занятые даты |
| `DEPLOY.md` | Инструкция по запуску Telegram Mini App |

## Быстрый старт

1. Загрузить все файлы и папки с фото на GitHub Pages или Vercel
2. Создать бота через @BotFather
3. Прописать URL сайта командой `/setmenubutton`

Подробности — в [DEPLOY.md](DEPLOY.md).

## Панель хозяина

Открыть `admin.html` → пароль по умолчанию `owner2024`.
Настройки сохраняются в localStorage и автоматически применяются на сайте.
